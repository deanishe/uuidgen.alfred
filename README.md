# UUIDGen for Alfred Workflow

Simple Alfred Workflow to generate UUIDs.

![Screenshot][screen1]

## Usage

Place new UUID at Cursor:

	uuid [Enter]
    
Place new UUID on Clipboard:

	uuid [CMD-Enter]
	
	
Place new UUID at Cursoer without newline:

	uuid [ALT-Enter]
